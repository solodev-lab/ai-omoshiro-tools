# Galaxy Cycle画面 星ビジュアル仕様書

## 概要
BlueYardサイトを参考にしたゴールド基調の統一カラーパレット。
MaskFilter.blur は Impeller レンダラーでポップアーティファクトを起こすため全面禁止。
グロー効果は全て多層半透明円で実現する。

## 星コア描画

- **透過**: alpha 0.7（背景が透ける）
- **RadialGradient**: ハイライト（左上オフセット）→ coreCol → 暗いエッジ
  - stops: [0.0, 0.5, 1.0]
  - ハイライト: coreCol + 白55%混合
  - エッジ: coreCol + 黒30%混合、alpha × 0.6
- **スペキュラドット**: dotR > 2.0 でフェードイン、半径 dotR×0.18、白 alpha最大0.3

## グロー仕様

### 全て共通ルール
- **MaskFilter.blur 使用禁止**（Impellerポップアーティファクト防止）
- 多層半透明円で代替
- alpha は二次曲線（t²）で外側から中心に向けて滑らかに増加

### 通常グロー（全reading星）
| パラメータ | 値 |
|---|---|
| 層数 | 8 |
| 半径範囲 | gR×1.8（外側）→ gR×1.0（内側） |
| gR | baseSize × 1.2 |
| alpha基準 | 大アルカナ: 0.15 × breath / 小アルカナ: 0.35 × breath |
| 色 | glowCol（属性/惑星色） |

### 大アルカナ追加グロー（Full Moon以外）
| パラメータ | 値 |
|---|---|
| 層数 | 8 |
| 半径範囲 | dotR+11（外側）→ dotR+4（内側） |
| alpha最大 | 0.07（Full Moonの半分） |
| 色 | `rgb(255, 240, 192)` ウォームゴールド |

### Full Moon専用グロー
| パラメータ | 値 |
|---|---|
| 層数 | 8 |
| 半径範囲 | dotR+22（外側）→ dotR+8（内側） |
| alpha最大 | 0.14 |
| 色 | `rgb(255, 240, 192)` ウォームゴールド |
| 判定 | 月に1日のみ（正午から±12時間以内） |

### Stellaコア（中央固定星）
| パラメータ | 値 |
|---|---|
| 層数 | 20 |
| 半径範囲 | 42px（外側）→ 4px（内側） |
| alpha最大 | 0.25 |
| 色 | solaraGold → solaraGoldLight（lerp） |

## 背景

### メイン背景グラデーション
- RadialGradient center(0, -1), radius 1.1
- `#0F2850` → `#080C14`

### 星雲グラデーション（奥行き演出）
- 5つの星雲（位置は毎回ランダム）
- 中央以外の4つは色も8色パレットからランダム選択（金色以外の寒色系）
- 中央のゴールドのみ色固定（位置は±0.15のジッター）

| 基準位置 | 色パレット | alpha | radius |
|---|---|---|---|
| 左上 (-0.8, -0.6) ±0.4 | ランダム（寒色系） | パレット依存 | 0.8 |
| 右下 (0.7, 0.8) ±0.4 | ランダム（寒色系） | パレット依存 | 0.7 |
| 中央左下 (-0.15, 0.15) ±0.3 | ランダム（寒色系） | パレット依存 | 0.6 |
| 左下 (-0.7, 0.6) ±0.4 | ランダム（寒色系） | パレット依存 | 0.65 |
| 中央 (0, 0) ±0.15 | `#F9D976` ゴールド固定 | 19% | 0.45 |

星雲色パレット（8色）:
- `#402060` 紫 38% / `#102850` 深青 31% / `#102850` ダーク青 25%
- `#80D0F0` 水色 15% / `#304060` スチールブルー 19%
- `#502060` バイオレット 21% / `#60A0B0` ティール 16% / `#203050` ネイビー 19%

### 背景星（twinkle）
- 30個の白い小ドット（半径0.5〜1.5px）
- 位置は毎回ランダム（bgSeedで決定）
- MaskFilter.blur なし
- sin波で明滅（4〜10秒周期、alpha 0.1〜0.8）

---

## ランダム化
- `_bgSeed = DateTime.now().microsecondsSinceEpoch`（画面を開くたびに更新）
- 背景星の位置・サイズ・明滅タイミング: bgSeedで決定
- 星雲の位置: bgSeedで決定（基準位置からジッター）
- 星雲の色（中央以外）: bgSeedで8色パレットからランダム選択
- ホットリロード(r)では変わらない。ホットリスタート(R)で変わる

---

## Breathingアニメーション
- グローのalphaにのみ影響（星サイズには影響しない）
- `breath = 0.7 + 0.3 × sin(now / period + phase)`
- 各星ごとに位相(phase)と周期(period)が異なる（決定的PRNG）

---

## 禁止事項
- MaskFilter.blur — Impellerでポップする
- 星サイズの動的変化 — breathやscaleで変えない
- z-ソートの昇順 — 降順（rz大→小）にすること
- リング描画（回転楕円・Strokeリング）— 全て廃止済み
