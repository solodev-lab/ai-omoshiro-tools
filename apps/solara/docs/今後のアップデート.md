### 2 — Daily Omen Button + Dominant Fortune Overlay（2026-04-22 時点、本番挙動に切替済み）
実装済み:
- `widgets/omen_button.dart` — 今日のタップボタン（呼吸グロー、ランダム文言）
- `utils/omen_phrases.dart` — 10フレーズ×4CTA のランダム組み合わせ
- `widgets/dominant_fortune_overlay.dart` + `widgets/fortune_overlays/` — 5カテゴリ演出（Solara世界観に統一）
- `sanctuary/sanctuary_reset_hour_picker.dart` — 1日の開始時刻設定 (0-23時)
- リセット時刻考慮: `SolaraStorage._logicalTodayKey()`

**現在の仕様:**
- Map画面を開くと1日1回、Omen Button が画面下に出る
- タップすると当日ホロスコープ最高カテゴリの演出（4.0秒）が発動
- 1日1回は Sanctuary の「1日の開始時刻」を基準に判定（デフォルト 00:00）
- `_debugAlwaysShowOverlay = false` / `_debugCycleOverlayKinds = false` で本番運用中

**将来の改善余地:**
- 低スペック端末向けに粒子数を動的調整（デバイス性能検知）
- オーバーレイ中のハプティックフィードバック追加

- Omen Button タップ音（SFX）と振動
- フレーズ選出を日付seedで固定（1日中同じ文言にする案）
