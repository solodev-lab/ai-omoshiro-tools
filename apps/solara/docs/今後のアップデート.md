### 2 — Daily Omen Button + Dominant Fortune Overlay（2026-04-22 時点、本番挙動に切替済み）
実装済み:
- `widgets/omen_button.dart` — 今日のタップボタン（呼吸グロー、ランダム文言）
- `utils/omen_phrases.dart` — 10フレーズ×4CTA のランダム組み合わせ
- `widgets/dominant_fortune_overlay.dart` + `widgets/fortune_overlays/` — 5カテゴリ演出（Solara世界観に統一）

**現在の仕様:**

- `_debugAlwaysShowOverlay = false` / `_debugCycleOverlayKinds = false` で本番運用中

**将来の改善余地:**

### 4 — 地図の多言語対応（2026-04-23 Jawg + Smart ハイブリッド実装済み）

**実装済み:**
- Jawg Maps ラスタタイル（`?lang=xx` で多言語ラベル切替）
- Smart ハイブリッドスタイル（言語ホーム圏は OSM HOT 無料、圏外は Jawg 有料）
- CF Worker プロキシ（トークン秘匿、edge cache 24h、レートリミット 600req/分）
- スマートデフォルト言語（端末ロケール検出、日本語端末=ja、それ以外=en）
- 米国ターゲット Phase 1 完了（デフォルト smartLight、bbox にハワイ・プエルトリコ追加）

**現在のコスト構造:**
- Jawg 無料枠: 25k map views/月
- 実対応 MAU: 日本語市場 30〜100人、英語市場 100〜500人（OSM でカバー率が異なる）

**超えたときの選択肢:**

- Jawg 有料プラン: €250/月で 100k views（MAU 数百〜千人）
- Protomaps 自前ホスト: Cloudflare R2 に PMTiles 配置、月 $1〜$5 で実質無制限。
  ただし vector_map_tiles のシェーダ互換性問題が解決するまで Flutter 実装ブロッカーあり
- MapTiles API $25/月（1M tiles、英語のみ対応、日本語非対応）

**未実装（将来）:**
- Smart Dark モード（ColorFilter invert 衝突問題を解く必要あり）
- Flutter i18n（UI の英語化）— リリース直前までまとめて実施
- Phase 2: Smart ハイブリッドの bbox 精密化（欧州の主要国別判定等）
- FORECAST画面からHORO画面への移動機能、伴い、HORO画面の日付指定方法変更
