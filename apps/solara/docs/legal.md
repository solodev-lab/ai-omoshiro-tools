## 1. プライバシーポリシー

### 掲載場所
- **ウェブ上**: solodev-lab.com/privacy（URLをストアに登録）
- **アプリ内**: Sanctuary画面 → 設定 → Privacy Policy（現在空実装）

### 記載必須項目

1. **収集する情報**
   - ユーザーが入力する情報: 名前、生年月日、出生時刻、出生地
   - 自動収集する情報: なし（現時点）
   - Firebase導入後: デバイス情報、利用状況、クラッシュログ

2. **情報の使用目的**
   - 占い結果の計算に使用
   - アプリの改善（Firebase Analytics）
   - クラッシュの検知と修正（Crashlytics）

3. **情報の保存場所**
   - 端末内のみ（SharedPreferences）
   - サーバーには送信しない（現時点）
   - Firebase導入後: Googleのサーバーに匿名データが送信される

4. **第三者提供**
   - 行わない（現時点）
   - Firebase導入後: Google（Analytics/Crashlytics）

5. **データの削除**
   - アプリをアンインストールすれば全データが削除される
   - アプリ内からデータ削除する機能（将来実装推奨）

6. **お問い合わせ先**
   - メールアドレスを記載

## 2. 占いアプリ特有の免責事項

### ストアでリジェクトされる表現

| ❌ NG表現 | ✅ OK表現 |
|----------|----------|
| 「当たる占い」 | 「星の配置に基づく占い」 |
| 「あなたの運命が分かる」 | 「あなたの星のエネルギーを読み解く」 |
| 「100%的中」 | 記載しない |
| 「病気が治る方角」 | 記載しない（医療行為と誤解される） |
| 「この方角で投資すれば儲かる」 | 記載しない（金融助言と誤解される） |

### アプリ内に表示する免責文言

以下の文言を星読み結果画面に表示する：

```
本アプリの占い結果は、天文学的データに基づく
エンターテインメントコンテンツです。
医療、法律、金融等の専門的な助言として
利用することはできません。
重要な判断については、専門家にご相談ください。
```

**表示場所の候補:**
- 星読み結果画面のフッター
- 初回起動時の同意画面
- Sanctuary画面の設定 → About内

## 3. Google Play 特有の注意

### コンテンツレーティング
- 占いアプリは「PEGI 3」または「全年齢」で通ることが多い
- ただし「運勢」「運命」等の表現は「迷信的コンテンツ」として13歳以上に分類される可能性
- **推奨: 13歳以上（Teen）で申請する**

### データセーフティセクション
Google Play にはデータセーフティの質問がある。正直に回答する：

| 質問 | 回答 |
|------|------|
| データを収集するか？ | はい（名前、生年月日等） |
| データを共有するか？ | いいえ（現時点） |
| データは暗号化されるか？ | いいえ（SharedPreferences） |
| データの削除を要求できるか？ | はい（アンインストール） |

### 広告表示時の追加対応（将来）
- AdMob導入時は「広告を含む」と明記
- パーソナライズ広告のオプトアウト機能が必要
- 占いアプリは「ヘルスケア」カテゴリ扱いされることがあり、パーソナライズ広告が制限される場合がある

## 4. 特定商取引法表記（課金導入時）

課金機能を実装する場合、以下を記載する：

- 販売業者名
- 運営責任者名
- 所在地
- 連絡先（メール）
- 販売価格
- 支払い方法
- 返品・キャンセルポリシー

**注意:** 個人開発者でも課金がある場合は特商法の表記が必要。
住所を公開したくない場合はバーチャルオフィス等を利用する。
