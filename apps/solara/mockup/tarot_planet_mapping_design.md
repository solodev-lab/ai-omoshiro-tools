## ホロスコープ アスペクトパターン検出仕様（確定）

### 検出ルール
- パターン構成惑星にネイタル10天体を使用（個人天体5に限定しない）
- クロスチャート（2重・3重）では、構成3惑星のうち**最低2つがネイタル**であること
- 構成惑星のうち**最低1つが個人天体**（☉☽☿♀♂）であること
- トランジットのみ・プログレスのみのパターンは除外（個人に関係ないため）

### パターン検出オーブ
Solara標準オーブ値はSPEC.mdに一元管理。Sanctuary Aspect Orbsオーバーレイからユーザー変更可能。

> 設計根拠: 4°→実測で多すぎ、2°→ほぼ検出なし。3°基準で月1〜2回のレア感。

### パターン接近アニメーション仕様（確定）

CSSアニメーション（opacity + filter）のみ使用。GPU compositing処理でCPU負荷ほぼゼロ。
ブラウザタブ非表示時はCSS animationが自動停止するため、バッテリー消費なし。

| 状態 | 演出 | 速度 | 詳細 |
|------|------|------|------|
| 3日前〜12時間前 | パターン枠がゆっくり明滅 | 4秒周期 | opacity 0.3↔0.8、ease-in-out |
| 12時間以内 | 2色交互に切替 | 2秒周期 | 金(#F6BD60)↔白金(#E8E0D0) |
| パターン成立中 | 常時点灯＋微かな光の揺らぎ | 3秒周期 | opacity 0.4↔0.7、背景も連動点滅 |

#### デザイン要件
- ease-in-outで滑らかに遷移（パチパチしない上品な明滅）
- 色は金(#F6BD60)↔白金(#E8E0D0)の2色（Solaraブランドカラー）
- SVG filterのblurで光がにじむ効果を付与
- 常時再描画は発生させない（CSSプロパティ変更のみ）

## ホロスコープ画面 確定仕様（v2 — 2026-03-28 確定）

### ブランディング
- ページ上部のロゴテキスト: なし（削除済み）
- **ウォーターマーク**: チャート中央やや上（42%位置）に「SOLARA」透かし表示
  - font-size: 14px / font-weight: 700 / letter-spacing: 4px
  - color: rgba(246,189,96,0.18) / pointer-events: none / z-index: 2（SVG前面）
- サブタイトル「powered by astronomy-engine」: 不要（MITライセンス、UI表示義務なし）

### チャートモード
- **1重円（Single）**: ネイタルのみ
- **2重円（Double）**: ネイタル＋トランジット
- **3重円（Triple）**: ネイタル＋プログレス＋トランジット
- 円切替時: 全フィルタ・個別チェックを初期状態にリセット

### 天文計算
- **ライブラリ**: astronomy-engine v2.1.19（ブラウザ内計算、CDN読込）
- **方式**: 地心（Geocentric）
  - 月: `Astronomy.EclipticGeoMoon(date).lon`
  - 太陽: `Astronomy.SunPosition(date).elon`
  - 惑星: `Astronomy.GeoVector(body, date, true)` → `Astronomy.Ecliptic(geo).elon`
- **セカンダリプログレッション**: 1日 = 1年

### カラースキーム
| 要素 | カラー |
|------|--------|
| ネイタル惑星 | #FFD370（金色） |
| トランジット惑星 | #6BB5FF（青色） |
| プログレス惑星 | #B088FF（紫色） |
| ブランドカラー金 | #F6BD60 |
| ブランドカラー白金 | #E8E0D0 |
| 星座記号フォント | #B49774（牡牛座色で統一） |
| 星座枠背景 | 各星座固有カラー（ZODIAC_COLORS配列） |

### 星座帯
- 12星座それぞれの固有色で枠背景を表示
- 星座記号のフォント色は全て `#B49774` で統一
- **タップでツールチップ表示**: 日本語名称（例: ♈ → 牡羊座）
  - 上部の星座: ツールチップを下に表示
  - 下部の星座: ツールチップを上に表示
  - 2秒後に自動消滅、別の星座タップで即切替

### トランジット入力
- 日付・時刻入力欄
- 場所セレクター: 東京/大阪/岐阜/カスタム（緯度経度直接入力）
- デフォルト: 現在日時・東京

### アスペクト
- メジャー5種 + マイナー3種（Solara標準オーブ値はSPEC.md参照）
- 2重円: N-Tアスペクトのみ表示
- 3重円: N-P（点線）＋N-Tアスペクト表示
- 各アスペクト行に個別ON/OFFチェック（☑/☐）
- チェック切替時はSVG線のみ再描画（一覧スクロール位置保持）

### アスペクトリスト（折りたたみ式）
- `analysis-container` 形式の折りたたみパネル（ヘッダークリックで開閉）
- 1重・2重円: タブなし、タイトルにモード名＋カウント表示
- 3重円: **N↔T Transit** / **N↔P Progressed** のタブ切替（各タブにカウント表示）
- 初期状態: 展開

### ASPECT FILTERパネル（折りたたみ式）
- `analysis-container` 形式の折りたたみパネル
- ヘッダー: 「⚙ ANALYSIS」＋ ▼/▲ トグル
- フィルタ6セクション（A〜F）とRESETボタン
  - **A**: アスペクト性質（ソフト/ハード/中立）— トグル式
  - **B**: 運勢カテゴリ（癒し/金運/恋愛運/仕事運/コミュニケーション）— 排他式
  - **C**: 惑星グループ（個人天体/社会天体/世代天体）— トグル式
  - **D**: アスペクト個別（6種チップ）— トグル式
  - **E**: オーブ強度（タイト2°以内/標準）— トグル式
  - **F**: アスペクトパターン（グランドトライン/Tスクエア/ヨッド）— 排他式
    - 選択時: 全アスペクトのチェックを外し、パターン構成アスペクトのみ再チェック
- **UPCOMING PATTERNS**: Fセクション直下に表示（タブ切替廃止）
  - 成立中パターン: ✔ 成立中
  - 今後の予測: 最大5件、時間/日数＋日付表示
  - パターンなし時は非表示

### パターンポリゴン描画
- アクティブパターン: 半透明塗りつぶし＋実線枠（stroke-width:2）
- 予測パターン（96時間以内）: ゴースト表示（点線枠、薄い塗り）
- グランドトライン色: #26D0CE
- Tスクエア色: #FF6B6B
- ヨッド色: #B088FF

### F欄枠アニメーション（CSS-only, GPU composited）
- approaching（96時間以内）: 枠＋背景が4秒周期で点滅
- imminent（12時間以内）: 枠＋背景が2秒周期で点滅
- active（成立中）: 枠＋背景が3秒周期で点滅
- border全辺が均一に点滅（!importantで:last-child上書き）
- padding-bottom確保（:last-childでも余白維持）

### パターン完成予測
- 予測期間: 60日
- ネイタルペア条件: 最低1つが個人天体
- スキャンオーブ: 3°
- 結果はhoursUntil順にソート、最大5件表示

### レスポンシブ対応

#### デスクトップ（幅 > 900px）
- 左右2カラムレイアウト（左: チャート600px / 右: パネル群）
- 左カラム: sticky配置（スクロール追従）
- SVG: viewBox 600x600、width:100%レスポンシブ
- ボトムシート: 非表示

#### モバイル（幅 ≤ 900px）
- 1カラム縦積みレイアウト
- 右パネル（BIRTH DATA等）: 非表示（ボトムシートで代替）
- **ボトムシート（Google Maps風）**:
  - position: fixed / bottom: 0 / border-radius: 16px 16px 0 0
  - backdrop-filter: blur(12px)
  - **ドラッグ3段階**: min（52px、タブのみ）→ half（45%）→ full（85%）
  - ドラッグハンドル: 36x4px金色バー
  - **5タブ**: ⚙誕生 / ☾経過 / ☉天体 / ⚙絞込 / △相
  - 経過タブ: 1重円では非表示、2重/3重で自動表示
  - チャートモード切替時: アクティブタブのコンテンツを自動再構築
  - リサイズイベント対応（画面回転等）
  - body padding-bottom: 380px（チャートがシートに隠れない）
  - 初期状態: half
